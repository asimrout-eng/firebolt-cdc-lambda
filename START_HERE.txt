QUICK START:
-----------

1. Configure Credentials:

   macOS/Linux:
   cp .env.example .env
   nano .env    # Add your Firebolt credentials

   Windows (PowerShell):
   Copy-Item .env.example .env
   notepad .env

   Windows (CMD):
   copy .env.example .env
   notepad .env

2. Deploy:

   macOS/Linux:
   chmod +x scripts/*.sh
   ./scripts/deploy.sh

   Windows (PowerShell):
   .\scripts\deploy.ps1

   Windows (CMD):
   scripts\deploy.bat

3. Upload table keys:

   macOS/Linux:
   aws s3 cp config/tables_keys.json s3://fcanalytics/firebolt-migration/config/tables_keys.json

   Windows:
   aws s3 cp config\tables_keys.json s3://fcanalytics/firebolt-migration/config/tables_keys.json
